# Survival Game - Level 1
A simple 2D survival game built with Orthodox C++ and Raylib.

## How to Compile
Ensure you have Raylib installed on Ubuntu, then run:
g++ main.cpp -lraylib -lGL -lm -lpthread -ldl -lrt -lX11 -o run

## How to Play
- Use WASD or Arrow Keys to move.
- Collect the Gold coins.
- Avoid the Red enemy!
